To access the API Gateway from outside the cluster:

1. Get the service name:
   kubectl get svc -n {{ .Release.Namespace }} {{ template "common.names.name" . }}

2. Port forward to access locally:
   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ template "common.names.name" . }} 8080:8080

3. Access the API Gateway at:
   http://localhost:8080
